## Install packages or API:

Create .env file add this variables
```
DATABASE =
DATABASEURL = 
PORT = 
AWS_ACCESS_KEY_ID = 
AWS_SECRET_ACCESS_KEY = 
AWS_REGION= 
AWS_BUCKET_NAME= 
```
Pre-requiste you should have node installed 

```
$ cd server
```

```
$ npm run start
```
```
$ cd client
```
```
$ npm run dev
```
